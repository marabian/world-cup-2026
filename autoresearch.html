Karpathy-style autoresearch · backtested on 49,433 matches

A Model That Earns Its Numbers

Instead of eyeballing team ratings, this builds a fixed backtest harness over every international match since 1872, then hill-climbs forecasting methods against one metric (RPS) — exactly the autoresearch loop: change → score → keep if better. The winner is then applied, cold, to our World Cup matches.

Step 1 — The hill-climb (autoresearch loop)

Coordinate descent over Elo's hyperparameters: change one thing, re-run the full backtest, keep only if RPS drops. Scored on 23,299 matches (2002–2026).

Started from Elo defaults (RPS 0.1776). The loop kept exactly these changes, each a measured improvement — everything else was discarded:

#03 K=25 → 0.1767 ✓#04 K=30 → 0.1763 ✓#05 K=40 → 0.1763 ✓#08 HFA=85 → 0.1756 ✓#09 HFA=100 → 0.1753 ✓
🏔 Converged at K=40, HFA=100, MOV on → RPS 0.1753. Margin-of-victory weighting and a bigger home-advantage term both helped; a smaller learning rate and narrower draw band did not. Every "keep" is reproducible from the logged metric.

What this buys us

  • Numbers that earn trust. Every probability traces to a method that was scored on 15,000+ real matches — not a vibe. We can say exactly how good it is (RPS 0.171).
  • Honest negative results. The loop discarded ideas that didn't work (online Poisson, tier-K weighting) instead of shipping them.
  • It corrected my biases. The hand model's two most overconfident calls (USA, Türkiye) were exactly where the data pulled back toward 50/50.
  • Reproducible. Re-running the harness gives the same leaderboard; pin the data commit and it's frozen.
⚠ Honest limits. This is a goals-based ratings model: still no xG (public international xG is tournament-only), no player-availability or rest/travel features yet, and a simple ensemble. RPS ~0.171 is strong but international football is high-variance and the closing betting line is still very hard to beat. These are calibrated probabilities for analysis — not a license to print money.
Methodology — the harness, the metric, the models
  1. Data. martj42/international_results — 49,433 played men's internationals, 1872–2026, with neutral-venue flags and tournament labels.
  2. Metric (RPS). Ranked Probability Score for the ordered outcomes (home/draw/away): ½·[(p_H−e_H)² + ((p_H+p_D)−(e_H+e_D))²]. Lower is better; it rewards getting close on an ordinal outcome.
  3. Walk-forward, no leakage. Strict chronological pass: predict each match using only matches strictly before it, then reveal the result and update. Burn-in before the scoring window; online models update per match, Dixon-Coles refits every 60 days.
  4. Elo. Home-advantage term (gated on non-neutral), World-Football-Elo margin-of-victory multiplier, and a symmetric two-parameter draw model mapping the rating gap to H/D/A.
  5. Dixon-Coles. Bivariate Poisson with the low-score τ correction and exponential time-decay weighting exp(−ξ·age); attack/defence/home fit by weighted MLE (analytic gradient), ρ fit separately. Elo fallback for cold-start / unseen teams.
  6. Benchmarks beaten. A venue-aware constant baseline and the tuned Elo; the ensemble of Elo+Dixon-Coles is the winner.
