Step 1 вЂ” The hill-climb (autoresearch loop)

Coordinate descent over Elo's hyperparameters: change one thing, re-run the full backtest, keep only if RPS drops. Scored on 23,299 matches (2002вЂ“2026).

What this buys us

  • Numbers that earn trust. Every probability traces to a method that was scored on 15,000+ real matches вЂ” not a vibe. We can say exactly how good it is (RPS 0.171).
  • Honest negative results. The loop discarded ideas that didn't work (online Poisson, tier-K weighting) instead of shipping them.
  • It corrected my biases. The hand model's two most overconfident calls (USA, TГјrkiye) were exactly where the data pulled back toward 50/50.
  • Reproducible. Re-running the harness gives the same leaderboard; pin the data commit and it's frozen.
вљ  Honest limits. This is a goals-based ratings model: still no xG (public international xG is tournament-only), no player-availability or rest/travel features yet, and a simple ensemble. RPS ~0.171 is strong but international football is high-variance and the closing betting line is still very hard to beat. These are calibrated probabilities for analysis вЂ” not a license to print money.
Methodology вЂ” the harness, the metric, the models
  1. Data. martj42/international_results вЂ” 49,433 played men's internationals, 1872вЂ“2026, with neutral-venue flags and tournament labels.
  2. Metric (RPS). Ranked Probability Score for the ordered outcomes (home/draw/away): ВЅВ·[(p_Hв€’e_H)ВІ + ((p_H+p_D)в€’(e_H+e_D))ВІ]. Lower is better; it rewards getting close on an ordinal outcome.
  3. Walk-forward, no leakage. Strict chronological pass: predict each match using only matches strictly before it, then reveal the result and update. Burn-in before the scoring window; online models update per match, Dixon-Coles refits every 60 days.
  4. Elo. Home-advantage term (gated on non-neutral), World-Football-Elo margin-of-victory multiplier, and a symmetric two-parameter draw model mapping the rating gap to H/D/A.
  5. Dixon-Coles. Bivariate Poisson with the low-score П„ correction and exponential time-decay weighting exp(в€’ОѕВ·age); attack/defence/home fit by weighted MLE (analytic gradient), ПЃ fit separately. Elo fallback for cold-start / unseen teams.
  6. Benchmarks beaten. A venue-aware constant baseline and the tuned Elo; the ensemble of Elo+Dixon-Coles is the winner.
